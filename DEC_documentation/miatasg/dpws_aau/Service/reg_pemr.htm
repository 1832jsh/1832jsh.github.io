PYXIS Error Mask Register (ERR_MASK)
Address: 87.4000.8240
The ERR MASK is used to disable the logging and reporting of errors. On power-up, the default is for error logging to be disabled (ERR_MASK = 0). For each bit, a zero (0) disables the logging/reporting of that error, and a one (1) enables the corresponding error logging and reporting.
