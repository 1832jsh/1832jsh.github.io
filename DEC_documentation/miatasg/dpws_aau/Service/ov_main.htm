System Overview

The DIGITAL Personal Workstation a/au-Series system has the following features:
  • DC21164A (EV56) Alpha CPU running at 433, 500, or 600 MHz.
  • 2 and 4 MB SRAM Bcache (8 MB available later).
  • Data path width: 128-bits data plus 16-bits ECC.
  • 32 MB to 1.5 GB of ECC-protected Synchronous DRAM (SDRAM) main memory (SRAMs mounted on DIMMs).
  • 64-bit PCI bus.
  • ISA bus.
  • 1 MB flashROM firmware.
  • Enhanced IDE interface.
  • Universal Serial Bus (USB) with two external connector.
  • Eight diagnostic LEDs.
  • Mini-tower desktop enclosure.
  • EIDE controller for CD-ROM drive.
  • Ethernet: Fast Ethernet Lan controller for both 10- and 100-Mbps data rates.
  • CD-quality audio.
  • 300-Watt power supply.
  • Five option slots:
    • Slot 1. 32-bit PCI (half-size).
    • Slot 2. 32-bit PCI/ISA  combination (half-size).
    • Slot 3. 32-bit PCI/ISA combination (full-size).
    • Slot 4. 64-bit PCI/ISA combination (full-size).
    • Slot 5. 64-bit PCI (full-size).
  • Six drive bays:
    • One 3.5-inch front accessible (floppy diskette drive).
    • Three 5.25-inch front accessible devices (half-height [1.6-inch]).
    • Two internal 3.5-inch drives (low-profile [1.0-inch]).
  • Choice of operating system:
    • Microsoft Windows NT.
    • OpenVMS.
    • DIGITAL UNIX.
