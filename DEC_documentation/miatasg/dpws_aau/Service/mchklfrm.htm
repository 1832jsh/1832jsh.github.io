Machine Check Logout Frame

The following is the format of the Machine Check Logout Frame (MCLF) built by the console firmware for uncorrectable errors vectored through SCB entries 660 and 670.

The "r" in the table below indicates that the error can be retried when set.
The "s" in the table below indicates that this is the second error.

Machine Check Logout Frame: Header

 6 6                       3 3                         0
 3 2                       2 1                         0 offset
+-+-------------------------+---------------------------+ 
|r|          SBZ            |           bytes           | 000h
+-+-------------------------+---------------------------+
|       system offset       |          CPU offset       | 008h
+---------------------------+---------------------------+
|                           |        mcheck code        | 010h
+---------------------------+---------------------------+
|             shadow registers [14:8]<63:0>             | 018h
|                                                       |
+-------------------------------------------------------+
|              shadow register [25]<63:0>               | 050h
+-------------------------------------------------------+
|            PAL Temp registers [23:1]<63:0>            | 058h
|                                                       |
+-------------------------------------------------------+
