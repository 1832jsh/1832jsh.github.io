Scache Address Register (SC_ADDR)
Address: IPR CBU, FF.FFF0.0188
The SC_ADDR register is a read-only register. The address is loaded into this register every time the Scache is accessed if one of the bits in the SC_STAT register is not set. If an Scache tag or data parity error is detected, this register is locked until the SC_STAT is read.
