Updating Firmware

Note: If the system has "5.25 proto" AlphaBIOS installed, this procedure does not apply. You must reflash the system using APU rather than AlphaBIOS. Refer to the DIGITAL Intranet URL ftp://beer.eng.pko.dec.com/pub/miatamfg.

Copy the AlphaBIOS and POST over the Internet, using the following steps:

  1. Open the URL http://www.windows.digital.com/support/sysoft.htp.
  2. You will be at the Windows Enterprise Computing screen.
  3. Scroll down to the DIGITAL Personal Workstation heading and choose 433a.
  4. You will now be at the DIGITAL Personal Workstation 433a Support Files screen.
  5. Choose AlphaBIOS and POST.
  6. Save the file on your system.
  7. Unzip the archive. The zip file contains V5.25 (or later) AlphaBIOS firmware, V3.38 (or later) Power On Self Test (POST) firmware, and installation instructions for the DIGITAL Personal Workstation a-Series.
  8. From Windows NT copy the files miataab.rom and miatapos.rom onto a FAT-formatted floppy diskette.
  9. Select Shutdown from the Windows NT Security dialog box (Ctrl/Alt/Del).
  10. Select Shutdown and Restart from the Shutdown Computer dialog box.
  11. Enter setup by pressing F2.
  12. Insert the diskette into the workstation's diskette drive.
  13. Select Upgrade AlphaBIOS from the initial Setup screen.
  14. From the Upgrade Options dialog box select the POST code image. NOTE: When upgrading the POST image the old and new version information will be displayed as unknown. Version information will be added in a future release of AlphaBIOS. Select System Configuration from the setup screen after the machine is reset to confirm the new fimrware version.
  15. Once the POST code image has been updated, continue the update process by depressing the ESC key and selecting Upgrade AlphaBIOS from the Setup screen.
  16. From the Upgrade Options dialog box select the AlphaBIOS image. NOTE: When upgrading the AlphaBIOS image the new version information will not be displayed. Version information will be added in a future release of AlphaBIOS. Select System Configuration from the setup screen after the machine is reset to confirm the new firmware version.
  17. Power cycle the workstation.
