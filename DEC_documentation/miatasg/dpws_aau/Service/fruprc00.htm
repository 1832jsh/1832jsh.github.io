Cover Removal

To remove the system cover, refer to the figure and perform the following steps:

  1. Shut down the system and all peripheral devices.
  2. Unplug the power cord.
  3. Remove the system cover lock, if present.
  4. Loosen the three thumb screws (1) that fasten the system cover to the enclosure.
  5. Slide the cover back, then up and away from the enclosure.
