PCI Configuration

Information about all the PCI devices in the system is presented in Display System Configuration when you select PCI Configuration and press[ Enter].

The following information is displayed for each PCI device:
  • Device Name: Name and model of the device as recorded in the device’s firmware.
  • Device Type: Lists the function of the device in the system.
  • Physical Slot: Lists the actual PCI slot number to which the device is attached.
  • Revision: Revision level of the device.

PCI Device Detail

You also can get additional detail about any of the PCI devices listed by using the following procedure:
  1. Press [Enter] to allow selection in the device list.
  2. Select the device about which you want additional detail.
  3. Press [Enter] to display the detail.
  4. Press [Esc] to return to the table of PCI devices.
The following information is displayed for the selected PCI device:
  • Bus Number: Virtual PCI bus number.
  • Function Number: Represents the number assigned to a particular function on a multifunction device. For example, a combination Ethernet/SCSI controller would be listed twice, with the first function listed as function 0 and the other function listed as function 1.
  • Device Number: PCI bus device number.
  • Configuration Space Header: Displays the information in the selected device’s PCI configuration space.
