Diagnostic Check Register (DIAG_CHECK)
Address: 87.4000.3000
The DIAG_CHECK is used for diagnostic DMA writes to write a known ECC pattern into memory and to provide the ECC pattern on any PCI I/O read operation. This register provides the ECC that gets written to memory or appended to the I/O read operation if the USE_CHECK bit is set in the PYXIS_DIAG CSR. The primary use for this is to verify the error paths.
Reserved<31:8>RO N/A
DIAG_CHECK<7:0>RW For diagnostic DMA writes and PCI I/O reads, the DIAG_CHECK register provides the quadword ECC.
