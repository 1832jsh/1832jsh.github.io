PYXIS Syndrome Register (PYXIS_SYN)
Address: 87.4000.8300
The PYXIS syndrome registers contains two 8-bit fields containing the error syndrome bits. The data is only valid after an error condition. It is latched until the PYXIS error register is written to clear COR_ERR and UN_COR_ERR. The PALcode must save this register prior to clearing the error conditions. The state of this register is undefined at all times except when an error has been detected.
