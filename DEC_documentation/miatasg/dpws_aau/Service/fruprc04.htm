Memory DIMM Replacement

To replace DIMM memory modules, shut down the system, remove power, remove the system cover, refer to the figure, and perform the following procedure:

  1. Put on your ESD wriststrap.
  2. Remove the MLB and set it on an antistatic surface.
  3. Remove a DIMM by pressing open the two retaining levers (1) and then pulling the DIMM (2) straight out. (DIMMs do not use the angle-mounting that SIMMs do.)
  4. Install the new DIMM by first orienting the key notches, then inserting the DIMM straight into the socket.
  5. Press firmly down until both retaining levers engage the DIMM.
  6. Replace the MLB.
