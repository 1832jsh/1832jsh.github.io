PYXIS Error Register (PYXIS_ERR)
Address: 87.4000.8200
The PYXIS_ERR register is used to log information pertaining to an error condition detected in the PYXIS chip. All bits of the PYXIS_ERR, except the LOST bits will be locked until the PYXIS_ERR is cleared by a software write. The LOST bits will be set whenever the PYXIS_ERR is already locked and another error is detected.
