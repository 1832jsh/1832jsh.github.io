Firmware for the
DIGITAL Personal Workstation a/au-Series System

This section discusses firmware for all three operating systems supported by the DIGITAL Personal Workstation a/au-Series system: Microsoft Windows NT; OpenVMS; or DIGITAL UNIX.

If you are using the Microsoft Windows NT operating system, refer to the section on AlphaBIOS firmware. If you are using either the OpenVMS or DIGITAL UNIX operating system, refer to the section on SRM firmware.
