Exception Summary Register (EXC_SUM)
Address: IPR Ibox, Index=10C
This register contains various types of arithmetic traps that have occurred since the last time the register was written to (cleared). Any write to the EXC_SUM register clears bits <08:02>.
IOV <16>

If set, this bit indicates an F-box convert to integer overflow or integer arithmetic overflow.
INO <15> If set, this bit indicates a floating inexact error.
UNF <14> If set, this bit indicates a floating point underflow.
FOV <13> If set, this bit indicates a floating point overflow.
DZE <12> If set, this bit indicates a divide by zero.
INV <11> If set, this bit indicates an invalid operation.
SWC <10> If set, this bit indicates that software completion is possible.
