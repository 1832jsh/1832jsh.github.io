Memory Configuration

The DIGITAL Personal Workstation a/au-Series system supports up to three banks of memory. Each bank contains two identical DIMMs. To ensure a match, the POST (power on self test) code use the I2C bus to read the characteristics of each DIMM from a ROM on the DIMM. If identical pairs are not installed, the POST code may reject the DIMMs and flag the memory as unavailable.

Memory banks can be populated in any order. The POST code automatically maps the largest banks to the lowest addresses starting at 00.0000.0000. This prevents holes in the memory address space.
