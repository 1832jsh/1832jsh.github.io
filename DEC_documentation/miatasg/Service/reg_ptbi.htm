Scatter/Gather Translation Buffer Invalidate Register (TBIA)
Address: 87.6000.0100
A write to the TBIA register will result in the specified group of scatter/gather TLB TAGs to be marked invalid and unlocked.
Reserved<31:2> N/A
TBIA<1:0> A write to this register invalidates the scatter/gather Translation buffers.
Bits<1:0>Meaning
00No operation.
01Invalidate and unlock the TLB TAGs that are locked currently.
10Invalidate the TLB TAGs that are unlocked currently.
11Invalidate and unlock all of the TLB TAG entries.
